# Smart Lender

A Smart Lender is an intelligent digital lending system that uses technologies such as Artificial Intelligence (AI), Machine Learning (ML), data analytics, and automation to evaluate loan applications, assess creditworthiness, and approve or reject loans quickly and accurately. Unlike traditional lending processes, which often require extensive paperwork and manual verification, a Smart Lender streamlines the entire loan lifecycle through digital processes.Machine learning-powered web application to predict loan applicant creditworthiness using Decision Tree, Random Forest, KNN, and XGBoost classifiers.
